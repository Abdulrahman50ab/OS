# ABSAN-OS  
### Advanced Beautiful Simulated Advanced Operating System

![ABSAN-OS](screenshot.png)

A stunning **Operating System Simulation** built in Java Swing with modern UI, process management, PCB, scheduling, IPC, and more!

## Features
- Process Creation & Management
- Real-time Process Table
- Dispatch, Suspend, Resume, Block
- Inter-Process Communication (IPC)
- Kernel Configuration
- Beautiful dark gradient theme
- Custom PCB Viewer

## Tech Stack
- Java 17
- Swing GUI
- Maven



## How to Run
```bash
mvn exec:java
